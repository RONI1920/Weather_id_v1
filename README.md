# 🌤️ CuacaID

Halo! 👋 Ini adalah **CuacaID**, proyek portofolio sederhana yang aku buat untuk belajar bikin web dinamis pakai JavaScript. 

Fungsinya simpel aja: buat ngecek kondisi cuaca *real-time* di seluruh ibu kota provinsi yang ada di Indonesia. 

## ✨ Fitur-fiturnya apa aja?
Walaupun sederhana, aku nyoba masukin beberapa fitur seru di sini:
* **Cek Cuaca Langsung:** Tinggal pilih provinsi di *dropdown*, datanya langsung muncul (suhu, angin, kelembapan, dll).
* **Dark Mode (Tema Gelap):** Ada tombol 🌓 di pojok kanan atas buat ganti tema ke mode gelap atau terang. Enak buat mata!
* **Responsive Layout:** Kalau dibuka di HP tampilannya memanjang ke bawah, tapi kalau dibuka di laptop/PC tampilannya jadi "gendut ke samping" (2 kolom) biar nggak capek *scroll*.
* **Ada Animasi Loading:** Biar nggak bosen pas nunggu datanya diambil dari internet.

## 🛠️ Dibuat pakai apa?
Nggak pakai framework yang ribet-ribet, murni pakai yang *basic*:
* **HTML5** (buat kerangkanya)
* **CSS3** (buat ngatur desain, warna, animasi, sama *layout grid*)
* **Vanilla JavaScript** (buat ngambil data cuaca pakai fitur `Fetch API` & `Async/Await`)

## 🚀 Cara Pakainya
Kalau kamu mau coba jalanin proyek ini di komputer kamu sendiri, gampang banget:
1. *Download* atau *clone* *repository* ini.
2. Karena ini butuh narik data cuaca asli, kamu **wajib** punya API Key dari OpenWeatherMap.
3. Daftar gratis di [openweathermap.org/api](https://openweathermap.org/api).
4. Kalau udah dapet kodenya, buka file `script.js` terus ganti tulisan `YOUR_API_KEY` sama kode yang kamu dapet tadi.
5. Tinggal klik 2x file `index.html` buat buka di *browser*!
