💡 Untuk data cuaca real-time: daftar gratis di openweathermap.org → ganti YOUR_API_KEY di script.js
Fetch API & Async/Await — Mengambil data cuaca secara asinkron dari WeatherAPI tanpa menghambat performa halaman.
Object Mapping Strategy — Menggunakan pasangan Key-Value untuk menghubungkan ID Provinsi dengan Ibu Kota secara instan dan efisien.
Dynamic Content Injection — Memperbarui elemen DOM secara real-time, termasuk penyisipan gambar icon cuaca dinamis melalui URL.
Event-Driven Programming — Menggunakan listener onchange untuk memicu alur logika aplikasi segera setelah pengguna memilih wilayah.
Data Normalization — Memproses data JSON mentah menjadi informasi yang ramah pengguna (pembulatan suhu, format waktu, dan unit metrik).
Robust Error Handling — Menggunakan blok try/catch untuk memastikan aplikasi tetap berjalan stabil meskipun terjadi gangguan API.
